## Python Student Management System

A simple Python-based Student Management System developed as a university project at Jordan University of Science and Technology (JUST).

### Features
- Add new students
- Enroll students in courses
- Add and update grades
- Generate student reports (average, highest, lowest grade)
- Menu-driven console interface

### Requirements
- Python 3.x
- colorama
#*------------------------------------------------------------------------------------------------------------------------------------------------*
### How to Run
```bash
pip install colorama
python main.py
